### 前言
基于RBAC权限管理的学生实习管理系统，伪前后端分离，Bug极多

### 系统功能
- 用户管理：提供用户的相关配置
- 角色管理：对权限与菜单进行分配
- 菜单管理：已实现菜单动态路由
- 系统日志：记录用户操作日志与异常日志
- SQL监控：采用druid 监控数据库访问性能
- 接口管理：方便统一查看管理接口
- 部门管理：配置系统用户所属部门组织
- 公司管理：配置系统用户所属担任职务
- 字典管理：配置维护系统中较为固定的数据

### 技术栈
- SpringBoot
- MyBatis
- SpringSecurity
- MySql
- Druid
- Swagger
- Redis
- JWT
- Pear Admin Layui
